Booting

Will this work on my computer?

Most modern PCs can boot a NIXKEY drive through the boot menu. The main exception is Secure Boot blocking some distros, which is why every order includes the quick-start card explaining the common fix.

Persistence

Can I save files and settings?

Yes for the persistence-ready listings. Those builds are intended to keep Wi-Fi details, browser bookmarks, downloads, and working files between sessions.

Beginners

What should I buy if I am new to Linux?

Linux Mint, Ubuntu, and Fedora are the clearest starting points. The catalog marks those listings as beginner-friendly and explains what kind of user each one fits best.

Independence

Is NIXKEY affiliated with distro maintainers?

No. NIXKEY is an independent storefront that prepares bootable media for open-source distributions. Distro names are used descriptively so customers know what is installed on each drive.

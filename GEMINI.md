# Project Context: NIXKEY

## Overview
NIXKEY is a premium brand for bootable Linux USB drives, focusing on a "Safe Onramp" for beginners and a "Tinkerer's Toolkit" for experienced users.

## Brand Identity
- **Accent System**: Teal and Orange.
- **Tone**: Dark, professional "Senior Software Engineer" tone for all copy.

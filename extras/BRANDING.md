# NIXKEY Brand Guidelines

## 1. The "Cyber-Industrial" Palette
A high-contrast, modern tech palette inspired by brushed metal, terminal screens, and cinematic lighting.

| Color Name | Hex Code | Purpose |
| :--- | :--- | :--- |
| **Nix Teal** | `#00CED1` | Primary Accent. Represents code, electricity, and precision. Used for the "KEY" in the logo, links, and primary buttons. |
| **Forge Orange** | `#FF8C00` | Highlight/Alert. Represents the "spark," activity, and warmth. Used for sale badges, notifications, and secondary accents. |
| **Gunmetal Grey** | `#2F4F4F` | Structure/Base. Represents the physical metal drive and the hardware. Used for footers, cards, and secondary backgrounds. |
| **Terminal Black** | `#121212` | Background. A deep, off-black for maximum contrast without the harshness of pure #000000. |
| **Steel White** | `#F5F5F5` | Text. Sharp, clean, and legible. |

## 2. Typography
*   **Headings:** `Inter`, `Roboto`, or `System UI` (Bold, Modern Sans-Serif)
*   **Monospace (Code):** `Fira Code`, `JetBrains Mono`, or `Courier New` (For the "NIX" or specific tech elements)

## 3. Logo Concepts (See `branding_preview.html`)

### Concept A: "The Root Directory"
**`NIX / KEY`**
*   **Idea:** Uses the forward slash `/`, the most iconic symbol in Unix/Linux file systems (the root directory).
*   **Style:** Minimalist, structural.

### Concept B: "The Prompt"
**`>_ NIXKEY`**
*   **Idea:** Evokes the command line prompt. It says "This is a tool for power users."
*   **Style:** Retro-modern, hacker chic.

### Concept C: "The Cursor"
**`NIX_KEY`** (with a blinking cursor on the underscore)
*   **Idea:** Represents the active state of a terminal.
*   **Style:** Dynamic, alive.

01 // Quick Start

  1. Plug in: Insert NIXKEY into a USB 3.0 port if available.
  2. Power on: Turn on the computer and tap the boot-menu key immediately.
  3. Select the key: Choose USB Storage Device, UEFI: USB, or the NIXKEY entry.
  4. Launch Linux: Select Try or Try / Install to reach the desktop.

If the machine goes straight back to Windows or macOS, move to section 03.

03 // If The USB Is Skipped

#1: SECURE BOOT If the drive is skipped, enter BIOS (F2/Del) and set Secure Boot to [Disabled].
#2: FAST BOOT / HIBERNATION Disable "Fast Boot" in BIOS and Windows. Windows "Shut Down" is often a hidden Hibernate that locks the drive.
#3: UEFI VS LEGACY (CSM) Most NIXKEYs use UEFI. If your drive doesn't appear, try toggling CSM/Legacy Support in BIOS.

04 // Screen & Display Fixes

Black Screen? Press e at boot menu, find quiet splash, replace with nomodeset, then F10.

Tiny Icons? Search Settings for "Display" → "Fractional Scaling" (Set to 150% or 200%).

05 // First 5 Minutes

Once the desktop appears, do these in order:

  1. Join Wi-Fi: Click the network icon near the clock and connect.
  2. Open the browser: Launch Firefox or LibreWolf to confirm the web works.
  3. Check the clock: Open time settings and enable automatic time if needed.
  4. Run updates: Open Update Manager or use the package command in section 06.

06 // Install Software

08 // Safe Use

SHUTDOWN PROTOCOL When you are finished, use Power Off from the system menu and let the desktop shut down fully before removing your NIXKEY.

Drive Safety: NIXKEY runs from the USB and will not touch your Windows or Mac files unless you manually open or mount your internal drive.

09 // Permanent Install

Ready to move beyond the live session? Every NIXKEY includes an "Install" shortcut on the desktop.

  • Backup: Back up your hard drive first.
  • Dual Boot: You can install Linux alongside Windows if space allows.
